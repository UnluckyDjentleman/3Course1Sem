0) To install nodemailer
	npm install nodemailer
1) Last task:
	Firstly you should add user:
		npm adduser
	After registration you can check it
		npm whoami
	Then you should init package with the command
		npm init
	After that you set parameters, and publish it
		npm publish
	Move to your profile to npmjs.com/settings/*yourprofilename*/packages to check it
	To install your package
		npm install *yourpackagename* (locally)
		npm install *yourpackagename* (globally)
	To uninstall it
		npm uninstall *yourpackagename*
2) That's all) Good luck!
